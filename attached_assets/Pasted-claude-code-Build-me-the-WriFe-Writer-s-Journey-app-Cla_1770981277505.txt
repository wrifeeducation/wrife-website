### Marking Rubric Logic (Built Into Database)

**Automated Scoring for W1-W5 Activities:**
```
Rules stored in database:
- Multiple choice: 1 point per correct answer
- Drag-drop: 1 point per correct placement
- Fill-in-blank: 1 point per correct word (exact match or synonym)
- Sorting: 1 point per correctly sorted item
- Matching: 1 point per correct pair

Mastery threshold: 80% or higher
Status indicators:
- Green (✓): 80-100%
- Yellow (⚠): 70-79%
- Red (✗): 0-69%
```

**AI Scoring for W6 Creation Activities:**
```
Claude API analyzes pupil's sentence and returns JSON:
{
  "formula_adherence": 0-3,
  "grammar_accuracy": 0-2,
  "meaning_clarity": 0-2,
  "personal_connection": 0-1,
  "total_score": 0-8,
  "percentage": calculated,
  "mastery": boolean (>= 75%),
  "feedback": "Well done! Your sentence follows the formula perfectly...",
  "suggestions": ["Try adding an adjective before 'dog'", "Consider using a more specific place"],
  "word_class_analysis": [
    {"word": "My", "class": "determiner", "correct": true},
    {"word": "dog", "class": "noun", "correct": true},
    {"word": "runs", "class": "verb", "correct": true}
  ]
}
```

**PWP Weekly Assessment:**
```
Every Monday, system analyzes pupil's 5 sentences from previous week:

1. Formula Consistency Check (0-25 points):
   - Count: How many sentences used formula correctly?
   - Award: 5 points per correct sentence

2. Subject Variation Check (0-25 points):
   - Count unique subjects used
   - Penalty: -5 points for each repeated subject beyond 2nd use
   - Award: 25 points if 5 different subjects

3. Story Development Check (0-25 points):
   - Count story parts: beginning, middle, end
   - Ideal distribution: 2-2-1 or 2-1-2
   - Award based on balance

4. Independence Check (0-25 points):
   - AI hint requests: fewer = higher score
   - Time per sentence: faster = more confident
   - Revision count: some is good, too many may indicate struggle

Weekly Score: 0-100 points
Green = 80-100 (Excellent)
Yellow = 60-79 (On track)
Red = 0-59 (Needs support)
```

**Formal Assessment Rubric (4-Part):**
```
Stored in database as assessment_rubric table:

Part A (Recognition - 10 points):
- 10 multiple choice questions
- 1 point each
- Auto-scored by system
- Mastery: 8/10 (80%)

Part B (Application - 10 points):
- 5 sentences to analyze
- Each sentence scored 0-2 by AI:
  * 2 = Formula perfect, grammar perfect
  * 1 = Formula mostly right OR grammar mostly right
  * 0 = Neither correct
- Mastery: 8/10 (80%)

Part C (Creation - 5 points):
- Write 1 original sentence
- AI scores on 5-point rubric:
  * Formula correct (2 points)
  * Grammar accurate (1 point)
  * Personal/creative (1 point)
  * Complete sentence (1 point)
- Mastery: 4/5 (80%)

Part D (Explanation - 5 points):
- Pupil explains concept (recorded or written)
- AI or teacher scores:
  * Understands concept (2 points)
  * Can teach it (2 points)
  * Clear communication (1 point)
- Mastery: 4/5 (80%)

Overall Mastery:
- 3+ parts mastered = MASTERY (advance to next lesson)
- 2 parts mastered = NEARLY THERE (continue practice 1 week)
- 0-1 parts mastered = INTERVENTION (5-day support protocol)
```
